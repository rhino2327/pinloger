개인정보처리방침

시행일: 2026년 5월 23일  |  최종 수정일: 2026년 5월 23일

PINLOGER(이하 "서비스")는 이용자의 개인정보를 중요하게 여기며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 방침은 서비스가 수집하는 정보, 사용 목적, 보호 방법을 안내합니다.

1. 수집하는 개인정보 항목

수집 경로 수집 항목 필수 여부

Google 로그인 이메일 주소, 프로필 이름, 프로필 사진 URL 필수
카카오 로그인 카카오 계정 이메일, 닉네임, 프로필 사진 URL 필수

2. 개인정보 수집 및 이용 목적

  • 회원 식별 및 로그인 서비스 제공
  • 여행 일정·비용·메모 등 콘텐츠 저장 및 동기화
  • 초대 코드를 통한 여행 멤버 공유 기능 제공
  • 커뮤니티 탭 공개 일정 표시 (본인 동의 시)
  • 실시간 환율 정보 제공
  • 서비스 오류 분석 및 품질 개선

3. 개인정보 보유 및 이용 기간

항목 보유 기간
회원 계정 정보 회원 탈퇴 시까지 (탈퇴 후 즉시 삭제)
여행 일정·비용 데이터 여행 삭제 시 또는 계정 탈퇴 시 즉시 삭제
공개 커뮤니티 일정 본인이 비공개 전환 또는 삭제 시까지
오류·충돌 로그 수집 후 90일

4. 개인정보의 제3자 제공

서비스는 이용자의 개인정보를 원칙적으로 외부에 제공하지 않습니다. 단, 아래의 경우는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나 수사기관의 요청이 있는 경우

6. 이용자의 권리와 행사 방법

이용자는 언제든지 다음 권리를 행사할 수 있습니다.

  • 열람 요청: 본인의 개인정보 처리 현황 확인
  • 정정·삭제 요청: 잘못된 정보의 수정 또는 삭제
  • 처리 정지 요청: 개인정보 처리의 일시 정지
  • 계정 탈퇴: 앱 내 설정 → 프로필 → 탈퇴하기

권리 행사는 아래 이메일로 요청하시면 지체 없이 처리합니다.

7. 개인정보 보호 조치

  • 모든 데이터는 Firebase Firestore에 저장되며 TLS/SSL 암호화로 전송됩니다.
  • 비밀번호는 Firebase Authentication에 의해 단방향 암호화되어 저장됩니다.
  • 서버 측 API 키는 Firebase Cloud Functions에서만 사용되며 클라이언트에 노출되지 않습니다.
  • 정기적인 Firebase 보안 규칙 점검을 통해 비인가 접근을 차단합니다.

8. 자동 수집 정보 (쿠키 등)

서비스는 별도의 쿠키를 사용하지 않습니다. Firebase SDK는 세션 유지를 위해 기기 내 로컬 스토리지를 활용하며, 이는 서비스 제공에 필수적입니다.

9. 아동 개인정보 보호

서비스는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만으로 확인된 계정은 즉시 삭제 조치합니다.

10. 개인정보처리방침 변경

본 방침이 변경되는 경우 앱 내 공지 또는 이메일을 통해 7일 전에 안내합니다. 중요한 변경 사항은 별도의 동의 절차를 거칩니다.
